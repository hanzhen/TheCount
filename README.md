The Count
========

An iOS app for counting using flip digits.

My first iOS5/Storyboard project from '11.

![Screenshot](Screenshot.png)

## Features

* Multiple counters
* Counter naming
* Custom increment size
* Core Data to store history
* LEFT HAND MODE!
* an iAd (I was experimenting!)
